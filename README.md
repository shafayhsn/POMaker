# PO Maker - Ultimate Final

React + Vite local prototype for garment factory purchase order control.

## Features
- Compact ERP-style UI
- Dashboard, Purchase Orders, Suppliers, Active Jobs, Library, Settings
- Draft supplier masking
- PO approval and number generation
- Tax toggle with registered supplier default
- Grid-based PO builder
- Print and export actions
- localStorage persistence

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Vercel
- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist
