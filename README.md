# PO Maker V5

Compact React + Vite PO system prototype for garment factory workflow.

## Included
- Compact ERP-style UI
- Purchase Order builder with line grid
- Full mock suppliers and BOM library
- Draft supplier masking
- Approval lock
- PO number generation
- Tax toggle
- Print / export hooks
- localStorage persistence

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy to Vercel
- Framework Preset: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
